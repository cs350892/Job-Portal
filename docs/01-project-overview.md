# 01. Project Overview

Welcome to the Job Portal MERN Stack Project!

This project is a full-stack web application built using the MERN stack:
- **MongoDB** (Database)
- **Express.js** (Backend Framework)
- **React.js** (Frontend Framework)
- **Node.js** (Backend Runtime)

## Project Structure

```
Job-Portal/
├── backend/         # Node.js + Express backend (API server)
├── frontend/        # React.js frontend (client app)
├── docs/            # Project documentation (step-by-step guides)
├── .env.example     # Example environment variables for backend
├── .gitignore       # Git ignore rules
└── README.md        # Project overview and quickstart
```

## Key Features
- User authentication (JWT-based)
- Job posting and application system
- Resume upload and management
- Role-based access (job seekers, employers)
- RESTful API with error handling
- Clean, scalable codebase
- End-to-end integration (frontend ↔ backend)

## Who Is This For?
This guide is for **beginners and intermediate developers** who want to:
- Learn the MERN stack by building a real-world project
- Understand best practices for project structure and documentation
- Deploy and run a full-stack app locally

## How to Use This Documentation
- Each major topic/setup step is documented in its own file in the `docs/` folder
- Follow the files in order for a smooth setup and learning experience
